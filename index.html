Tags de Texto

^Voltar
  • h1>h6

    Tags no formato < h1 >,< h2 >..., que são usadas para escrever um texto em destaque, como um título ou subtítulo.

    Elas seguem uma ordem de tamanho de acordo com o número que está na frente do h, sendo o h1 o maior texto e o h6 o menor.
  • p

    Tags no formato < p >, que são usadas para escrever um parágrafo.

    Quando utilizadas elas automaticamente criam um espaço entre os outros textos, assim como os < h >
  • blockquote

    Tags no formato < blockquote >, que são usadas para escrever uma citação ou observação.

    Elas são espaçadas automaticamente quando utilizadas

Tags de Formatação

^Voltar
  • u

    Tags no formato < u >, que são usadas para formatar textos com underline.

    Exemplo: Olá, estou em underline.

  • i

    Tags no formato < i >, que são usadas para formatar textos com itálico.

    Exemplo: Olá, estou em itálico.

  • strong

    Tags no formato < strong >, que são usadas para formatar textos com negrito.

    Exemplo: Olá, estou em negrito.

  • mark

    Tags no formato < mark >, que são usadas para formatar textos com marcação, semelhante a um marca texto.

    Exemplo: Olá, estou em marcação.

  • sub

    Tags no formato < sub >, que são utilizadas para formatar textos que fiquem abaixo de outros.

    Exemplo: Olá, estou abaixo.

  • sup

    Tags no formato < sup >, que são utilizadas para formatar textos que fiquem em cima de outros.

    Exemplo: Olá, estou em cima.

  • small

    Tags no formato < small >, que são utilizadas para formatar textos que fiquem pequenos.

    Exemplo: Olá, estou pequeno.

Tags de Lista

^Voltar
  • ol

    Tags no formato < ol >, que são utilizadas para criar listas ordenadas.

    Exemplo:

    1. Uva
    2. Laranja
    3. Manga

  • ul

    Tags no formato < ul >, que são utilizadas para criar listas não ordenadas.

    Exemplo:

    • Uva
    • Laranja
    • Manga

  • li

    Tags no formato < li >, que são utilizadas para criar itens em listas.

    Exemplo:

    • Um item

Tags de Link

^Voltar
  • a

    Tags no formato < a >, que são utilizadas para criar links.

    Para gerar o link nessa tag é necessário usar o href.

  • hr

  • font/style

    Tags no formato < font >, que são utilizadas no CSS para definir cor, fonte e tamanho do texto.

    Como são utilizadas no CSS, o style pode ser usado no HTML.

    Para cor: Vermelho

    Para fonte: Times New Roman

    Para tamanho: 30px

  • del

    Tags no formato < del >, que são utilizadas para formatar textos como "deletados", incluindo um traço neles.

    Como em: Deletado

  • abbr

    Tags no formato < abbr >, que são utilizadas para definir abreviações de textos.

    Nelas são utilizadas o title para mostrar a definição completa da palavra abreviada.

    Como em: CSS
